吉他乐理(二) 和弦推导

发表于2020-12-05|更新于2020-12-05|吉他
|字数总计:3.5k|阅读时长:11分钟|阅读量:

和弦是学习吉他的一个关键，大多数的爱好者（例如我）在初期学习吉他时都是通过记忆和弦指型来演奏歌曲，也就是记型不记音，对于和弦的原理也不够清楚，不清楚为什么和弦要那样按，除了这种按法还有哪些按法，这也是吉他学习中的一个瓶颈。其实有更简单的方法帮助我们在吉他指板上推导出任意和弦，需要的理论知识包括三点：（1）吉他指板的音程关系；（2）和弦音的构成；（3）调。

吉他指板

标准调弦

吉他有六根弦，一般除非指弹的特殊调弦外，大多数的弹唱爱好者用到的都是标准调弦，从六弦到一弦的音分别为E、A、D、G、B、E，之所以是这六个音作为标准调弦的音和我们的手指构成有关。

我们可以发现从六弦到一弦，音程分别为纯四度、纯四度、纯四度、大三度、纯四度。除了三弦和二弦是大三度外，其他相邻琴弦之间都是纯四度，利用这个关系我们可以快速的推算出相邻琴弦的音。

指板音程

从每根琴弦的空弦音开始，按照全半音关系可以推算出指板上所有的音。

以六线谱为例，对应的视角是从上往下是一弦到六弦，其中包含很多规律：

  • 六弦上的任意一个音，与其正上方在五弦的音为纯四度关系，例如六弦 3 品为 G，则五弦 3 品为 C（G A B C），纯四度为二又二分之个全音，即两个全音加一个半音。
  • 以五弦的 3 品音为基础继续往上找，四弦的 3 品音为 F，六弦三品和四弦三品音程为七度音，因此六弦三品 G 的八度音为四弦的五品，即四弦三品向右移两品得到八度音 G。若以四弦的音开始向上找八度音，则会不同，因为三弦和二弦之间为大三度，所以此时在二弦上需要向右移三品，例如：四弦三品的 F，对应二弦的八度音为二弦六品，同理以三弦上的音为基础向上找一弦上的八度音时，也是如此。
  • 根据这种规律，我们可以发现只要记住了六弦和五弦上的音，我们就可以快速得到四弦和三弦的音，得到四弦的音，我们就可以推导二弦的音，而一弦和六弦的音组成是一样的，因此记忆指板利用这种方法只需记住六弦和五弦的音即可。

和弦音构成

和弦是有三个及三个以上的音按照一定的音程关系构成的一组音。因此在指板上推导和弦时，首先要知道这个和弦的组成音。

对于不同的调，组成调式的和弦也不相同，但都满足一级、四级、五级和弦是大三和弦、二级、三级、六级是小三和弦，七级和弦是减三和弦的特性。

在每个调中的一级和弦就是对应调名的和弦，例如 C 大调一级和弦就是 C 大三和弦，以 C 为根音，唱作 1，bE 调的一级和弦就是 bE 和弦，以 bE 作为根音，唱作 1。不同的调的第一个音都唱做 1，这是首调思维的优点，通过这个特性，我们知道一级和弦的组成音是 1，3，5，那么我们以 bE 作为根音，作为 1，在指板上找到它的1、3、5音即可得到 bE 和弦。

由于每一个和弦都会有不同的按法，可以从根音在某一弦上进行分类，除根音外的音肯定只能往上找，因为和弦根音是最低音。因此可以考虑分成几种情况，根音在六弦，根音在五弦（此时六弦不弹响），根音在四弦（五弦六弦不弹），根音在三弦也是可以的，但是由于和弦至少包含三个音，那就是说这三个弦刚好构成三和弦的组成音，从和弦色彩上不够饱满。

六弦为根

![image-20201205124954336](/Users/caojun/Library/Application Support/typora-user-images/image-20201205124954336.png)

X 大三和弦

以六弦上任意一品的音 X 为根音，看作 1，我们推导出以 X 为 1 的 1，3，5音，就得到了 X 大三和弦，这个 X 叫什么，那就是什么和弦，例如 X = bE，那这个和弦就是 bE 大三和弦。

（1）向右找

六弦任意品 X = .1，则向上找五弦为 4，向右找两品为 5，再向上找为 1，1的上方为 4，因此向左一品为 3，再向左一品为 b3，b3 的大三度为 5，因此 b3 上方二弦为 5，5 的上方为 1。

至此我们找到了六根弦上以 X 音为根音的大三和弦组成音，食指大横按 X 所在的品，中指按三弦，无名指五弦，小指四弦。神奇的是，这其实就是初学者熟悉的 F 和弦的指型，只不过现在是 X 和弦而已。回想一下，F 和弦是横按一品，六弦一品就是 F，此时 X = F，所以这种按法就是 F 和弦。通过这种指法我们可以在指板上任意滑动得到 X 和弦，得到的和弦名称只跟 X 的音有关，无需再去变换指法。

按照正常的思维，我们推导 F 和弦的按法，可能会习惯以 C 调为基础，在 C 调中 F = 4，F 和弦的组成音为 F = 4，A = 6，C = 1。然后我们去指板上找 F，A，C 这三个音。

这种推导方法的优点在于使用了首调式的思维进行推导，推导 X 和弦那就是推导 X 调的一级和弦，X 为根音，X = 1，因此其组成音就是1，3，5，至于组成音的位置我们通过调弦的特性和指板的音程关系就可以推算出来，而且无需变化！

（2）向左找

![image-20201205124954336](/Users/caojun/Library/Application Support/typora-user-images/image-20201205124954336.png)

以 X 为根音 1 ，设品位为 Y，向左上方找，我们可以知道五弦 Y 品位为 4，Y - 1品为 3，Y - 2 品为 b3，四弦 Y - 3 品为 5，三弦 Y - 3 品为 1，二弦 Y - 3 品为 3， 一弦 Y 品为 1。

除去 Y - 3 品的三个音的大横按，我们可以发现这就是个 G 和弦的按法，当 X = G 时，Y = 3，Y - 3 = 0，所以刚好不用大横按，因此就得到了 G 和弦。

X 小三和弦

现在我们很快就能得到任意的 X 大三和弦，只要你能记住六弦上的音，那么推算小三和弦就很简单了，其不同之处就是将三音 3 降半音得到 b3，图中用绿色标注的即是 b3，因此指法的改变就是将中指松开就行，这样就得到了 X 小三和弦，然后就会发现除去大横按，这不就是 Em 和弦的指型么，回想 Em 和弦的按法，按住四五弦的二品，此时 X = 六弦空弦音 E，原来 Em 和弦就是这样来的。

X 七和弦

在上述基础上继续改变构成音我们就能得到七和弦，以大三和弦为基础可以构成大大七和弦，大小七和弦，大大七和弦即1，3，5，7，大小七和弦即1，3，5，b7，因此我们想办法将其中一个音变为 7 或者 b7 就得到了 X 七和弦。

这六根弦的音中，六弦的根音不能变，五弦的 5 可以变，但是要变为 7 或者 b7 需要向右移动三个品位或四个品位，超出手指的范围，四弦的 1 可以变，而且只需向左移 1 个品位得到 7：

![image-20201205132249248](/Users/caojun/Library/Application Support/typora-user-images/image-20201205132249248.png)

只需在 X 所在品位大横按，中指四弦，无名指三弦，小指五弦，即可得到 X 大七和弦。将四弦的中指松开，即得到了 X 属七和弦。

同理，X 小大七和弦和 X 小小七和弦就是在 X 小三和弦的基础上同样的推导，指法更简单。

五弦为根

![image-20201205133057164](/Users/caojun/Library/Application Support/typora-user-images/image-20201205133057164.png)

以五弦上的任意一个音记为 X，作为根音快速推导 X 三和弦、七和弦。

X 大三和弦

（1）向右找

将 X 作为根音，设品位为 Y，看作 1，向上找其大三和弦的组成音1，3，5。根据弦之间的纯四度和大三度关系，我们可以得到几个组成音：四弦的 Y + 2 品音 5，三弦的 Y + 2 品音为 1，这个和六弦为根中相同，因为这三根弦之间都是纯四度，而三弦和二弦是大三度，因此此时得到二弦的 Y + 2 品为 3，一弦 Y 品为 5，有这些音构成了 X 大三和弦。

这个 X 大三和弦的好像又有点相似，除去 Y 品的大横按，这个指法就是熟悉的 A 和弦，即按住二三四弦的二品，以五弦空弦音作为根音的 A 和弦，而五弦空弦音是啥？恰好就是 A，因此 A 和弦就是那样按。

通过这种指型，我们又可以在五弦上滑动调整 X，指型不变即可得到对应的 X 大三和弦。

同理，X 小三和弦就需要将二弦的 3 变为 b3，然后你会发现除去大横按这不就是 Am 和弦的指法了，只不过恰好五弦空弦音是 A，我们没有用到大横按而已。

X 七和弦的推导方法与六弦为根的方法相同，将三弦的 1 变为 7 和 b7 即可。

（2）向左找

![image-20201205133057164](/Users/caojun/Library/Application Support/typora-user-images/image-20201205133057164.png)

以 X 为根音，设品位为 Y，向左上方找，我们可以得到四弦 Y - 1 品为 3，Y - 2 品为 b3，三弦 Y - 3 品为 5，二弦 Y - 2 品为 1，一弦 Y - 3 品为 3。又可以发现将大横按的 Y - 3 品去除，或者将其看作第 0 品，则这就是一个熟悉的 C 和弦的指型。

四弦为根

![image-20201205135042191](/Users/caojun/Library/Application Support/typora-user-images/image-20201205135042191.png)

(1) 向右上方找

以四弦上的任意一个音为根音，记为 X，看作 1，快速推导其三和弦和七和弦。

以四弦 X 音为根音 1，品位设为 Y 品，根据纯四度关系得到 三弦 Y + 2 品为 5，三弦和二弦为大三度，因此得到二弦的 Y + 3 品为 1，一弦 Y + 3 品为 4，Y + 2 品为 3，由此得到了 X 大三和弦的组成音位置。然后会发现除去 Y 品的大横按，指法就是 D 和弦的按法，而我们熟悉的 D 和弦就是以四弦为根音的这种按法，四弦空弦音恰好就是 D！将一弦的 3 降半音得到 b3，构成了 X 小三和弦，这恰好就是 Dm 和弦的指法！

通过这种指型，我们又可以在四弦上滑动以任意音为根音即可得到 X 三和弦，七和弦。

三弦为根

![image-20201205144507715](/Users/caojun/Library/Application Support/typora-user-images/image-20201205144507715.png)

由于以三弦为根时，每一根弦都只能是和弦组成音，而且只有三个音，因此这种和弦指法很少见，但从理论上这两种按法也是对的。

总结

至此，我们知道了熟悉的 C 大调第一把位上的和弦为什么会那样按，因为根音所在弦不同：

以六弦为根向右上方找，我们得到了 F 指型的 X 和弦，向左上方找我们得到了 G 指型的 X 和弦。

从五弦为根向右上方找，我们得到了 A 指型的 X 和弦，向左上方找我们得到了 C 指型的 X 指型。

从四弦为根向右上方找，我们得到了 D 指型的 X 和弦。

将 X 替换为我们想要得到的和弦音名，使用任意一种指型就可以得到 X 大三和弦，只需将 3 音降半音就得到了 X 小三和弦，将非最低音的 1 替换为 b7 或 7 即可得到了 X 七和弦。

这些不同的指型得到的都是 X 和弦，其区别就在于和弦的根音位置是在哪一弦，而在每一弦上同一个音又会有多个位置，因此又可以衍生出成百上千的在不同把位上的 X 和弦。

因此当别人询问我们任意一个 X 和弦时，我们都可以通过这几个步骤快速推导出来 ：

（1）决定自己要以几弦为根音，记为 S 弦

（2）决定使用向右上方查找的指法还是向左上方查找的指法，此时可以确定指型为 T

（3）根据自己记忆的五弦和六弦的音，快速推导 X 音在 S 弦上的所在位置

（4）以 X 音为根音用 T 指型就可以得到 X 大三和弦

（5）如果 X 和弦是小三和弦，则将按 3 的手指左移一品

（6）如果 X 和弦是七和弦，则将按 1 的手指向左移一品或者两品
