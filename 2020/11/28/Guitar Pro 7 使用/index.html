Guitar Pro 7 使用

发表于2020-11-28|更新于2020-11-28|吉他
|字数总计:988|阅读时长:2分钟|阅读量:

Guitar Pro 7 mac 版界面：

和弦库

点击菜单栏中的音轨—显示和弦资料库，进入自己的和弦库，点击右上角的添加和弦图标就可以进入和弦库查找相应的和弦。

左侧的选项中可以调整想要的和弦类型及组成，以 C 和弦为例：

  • 命名中选择主音为 C，确定为 C 和弦

  • 选择类型中包括：

    M - C大三和弦，m - 小三和弦，aug - C 增三和弦，dim - C 减三和弦，7 - C 属七和弦，7M - C 大七和弦，依次类推确定和弦的类型是大小和弦或者七和弦或者挂留和弦等等。

  • 选择延伸音：在上述已选的基础上加延伸音，从而得到九和弦、十一和弦、十三和弦。

  • 变化音：可以先不选，意思是改变和弦中的某个音增加半度或者减少半度，例如 C 大三和弦，点击 5 度音/5+后，类型中自动变为aug，即升高五音后变为增三和弦。

  • 转位和弦中包括最低音和转位：

    • 保持最低音和和弦根音一样，调整转位则可得到对应的转位和弦。
    • 调整最低音为和弦外的音，如：C / A，则得到分割和弦。

  • 省略和弦内音：获取九和弦、十一和弦、十三和弦时可省去某个音。

选择完后右侧会自动筛选出所有的和弦。

和弦标注

将和弦标注在六线谱上方，方法如下：

点击文档-样式表单，找到页面及乐谱格式中最下方的和弦图表：

将显示在曲谱顶部的显示关闭，打开显示在曲谱内的显示。

指法推导和弦

使用 Guitar Pro 7 推算指弹谱中和弦：

在曲谱音节中添加对应的六根弦的品位按法，如二弦一品，三弦二品，四弦二品，假设我们不知道这个和弦是什么和弦，然后选中这个音节，接着点击左侧编辑面板中最下方的和弦图标，这时 Gp 7会自动根据我们指定的指法计算出对应的和弦，得到和弦 Am / E：

通过这种方式，可以将指弹谱中的按法填入 Gp 的曲谱中，然后能够快速得到和弦名称，作为一个训练和弦推导的方法，自己先根据指弹谱的按法推算和弦，然后对照 Gp 7的推算结果看是否一致。

调弦

特殊调弦方法：

双击下面的音轨名称，右侧会出现检测器面板，可以在菜单栏的预览打开显示检测器面板，选择右侧检测器面板的音轨，可以看到调音说明显示位置，点击设置图标自己调整即可。

右侧面板的演奏中可以调节演奏方式，如果需要模仿吉他琴弦拨响后的延音，则打开持续发声的开关。

乐谱

1.在六线谱中输入数字代表该弦弹响的品位，默认是四分音符，通过+/-可以调整音符时值

2.两拍之间插入一拍音节，使用 command 加 + 插入音节

反复符号

有两段旋律除了最后一部分不同之外，其他全部相同，这时乐谱上可以用到反复符号，在该旋律开始前快捷键 command + [ 加入前反复符号||:，在旋律结尾的第一部分后快捷键 command + ] 加入关闭重复序列符号:||：

然后在旋律不同的部分上方添加段落符号：

分别选择第一段落和第二段落：

这样旋律第一次以第一部分结尾完后回到旋律开始部分，第二次就不会再播放第一部分旋律，而是播放第二部分旋律。
