Guitar Pro 7 使用
发表于2020-11-28|吉他|工具•软件
Guitar Pro 7 mac 版界面： 和弦库点击菜单栏中的音轨—显示和弦资料库，进入自己的和弦库，点击右上角的添加和弦图标就可以进入和弦库查找相应的和弦。 左侧的选项中可以调整想要的和弦类型及组成，以 C 和弦为例： 命名中选择主音为 C，确定为 C 和弦 选择类型中包括： M - C大三和弦，m - 小三和弦，aug - C 增三和弦，dim - C 减三和弦，7 - C 属七和弦，7M - C 大七和弦，依次类推确定和弦的类型是大小和弦或者七和弦或者挂留和弦等等。 选择延伸音：在上述已选的基础上加延伸音，从而得到九和弦、十一和弦、十三和弦。 变化音：可以先不选，意思是改变和弦中的某个音增加半度或者减少半度，例如 C 大三和弦，点击 5 度音/5+后，类型中自动变为aug，即升高五音后变为增三和弦。 转位和弦中包括最低音和转位： 保持最低音和和弦根音一样，调整转位则可得到对应的转位和弦。 调整最低音为和弦外的音，如：C / A，则得到分割和弦。 省略和弦内音：获取九和弦、十一和弦、十三和弦时可省去某个音。 选择完后右侧会自动筛选出所有的和弦。
